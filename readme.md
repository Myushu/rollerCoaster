# Rolley Coaster by Aljosha Armitano

## Install dependencies
```
npm install
```

## Build the project
```
npm run build
```
The javascript file is : `dist/main.js`

## Run
```
npm run start < [file]
```
Some test files are available on the `test` folder.

### Run test
You can use a shell script to run all tests together.
```
npm run test
```
